NearbyFYI

An API for our cities and towns

We're like Google for cities and towns

Sort of. NearbyFYI is an API that provides access to structured data from documents generated by our local city and town government. If you've ever needed access to structured information and data from your city or town government, then you know how painful it can be.

How do we help you?

You won't spend time:

  • Creating screen scrapers
  • Setting up and maintaining web crawlers
  • Building document processing pipelines
  • Extracting text from HTML, PDF, Excel and Word documents
  • Denormalizing data sources
  • Performing named entity extraction with a local government model
  • Classifying documents using city and town terms
  • Archiving documents you find
  • Deploying and maintaining a search engine
  • Supporting an API

NearbyFYI generates structured city data from public, but difficult-to-access information within closed systems and their unstructured documents. We clean, classify, and extract semantic metadata from these documents and provide a simple, public API to access the structured data.

How is it different?

In contrast to current solutions, our open data platform is free for cities and towns and their constituents. We build low cost, software as a service solutions for municipalities that replace their existing tools. This makes the collection of data easier and then charge a fee to commercial users of the API. We believe that this new model flips what was a previous expense line item into an overall cost savings for cities and towns.

What do we want to do?
